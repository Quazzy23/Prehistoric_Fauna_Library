========================================================================
PREHISTORIC FAUNA LIBRARY
========================================================================

О ПРОЕКТЕ
Библиотека представляет собой автоматизированную систему сбора и 
систематизации палеонтологических данных о динозаврах. Система извлекает 
данные из Wikipedia (включая иерархию таксонов из Template:Taxonomy), 
проводит научную верификацию и формирует реляционную базу данных SQLite.

ТРЕБОВАНИЯ
- Python 3.13+
- Библиотеки: requests, beautifulsoup4
- Интернет-соединение (для сбора данных)


ИНСТРУКЦИЯ ПО ЗАПУСКУ

ШАГ 0: ПОДГОТОВКА (Если у вас ничего не установлено)
1. УСТАНОВКА PYTHON:
   - Скачайте Python версии 3.13 или выше с официального сайта python.org.
   - ВАЖНО: При установке ОБЯЗАТЕЛЬНО поставьте галочку 
     "Add Python to PATH" (Добавить Python в PATH). Без этого команды 
     не будут работать в консоли.
2. УСТАНОВКА VS CODE (Рекомендуется):
   - Скачайте Visual Studio Code (code.visualstudio.com).
   - Откройте в нем папку проекта "Prehistoric_Fauna_Library".
   - Это самый удобный способ: вы сможете видеть логи и базу данных 
     в реальном времени.
3. УСТАНОВКА БИБЛИОТЕК:
   Для работы системе нужны два дополнения. Откройте терминал в VS Code 
   или обычную командную строку (cmd) и введите: pip install requests beautifulsoup4
4. ПРОСМОТР БАЗЫ ДАННЫХ:
   Чтобы открыть итоговый файл .sqlite, вам понадобится вьюер:
   - Если вы в VS Code: Перейдите во вкладку расширений (Ctrl+Shift+X) 
     и установите SQLite от alexcvzz. После этого базу можно будет открыть.
     Чтобы открыть базу данных, нужно в VS Code нажать ПКМ на dinosaurs.sqlite - Open Database
     Снизу в Exlorer появится вкладка SQLITE EXPLORER. Ее надо развернуть. Там будет лежать dinosaurs.sqlite
     dinosaurs.sqlite тоже развернуть. Внутри будет 3 таблицы. Рядом с таблицей dinosaurs нажать show table
   - Если вы НЕ используете VS Code: Скачайте и установите программу 
     "DB Browser for SQLite" (sqlitebrowser.org). Это самый удобный и 
     бесплатный инструмент для работы с SQLite вне редактора кода.

ШАГ 1: НАСТРОЙКА
Перед запуском откройте файл /scripts/config.py. Там можно настроить параматры работы скриптов
В этом файле обязательно вместо заглушки "Your_email@email" впишите свой email адрес

ШАГ 2: ЗАПУСК (Два способа)
СПОСОБ А: Через VS Code (Самый простой)
1. Откройте файл /scripts/main_pipeline.py.
2. Нажмите иконку "Play" в правом верхнем углу.
СПОСОБ Б: Через командную строку (Windows CMD / PowerShell)
1. Перейдите через команду cd папку проекта (Prehistoric_Fauna_Library)
2. В открывшемся черном окне введите команду: scripts/main_pipeline.py


ЧТО ПРОИЗОЙДЕТ (PIPELINE):
1. Сбор имен и статусов родов со списочной страницы Wikipedia.
2. Сбор официальной геохронологической шкалы (ICS).
3. Глубокий парсинг страниц каждого динозавра:
   - Извлечение всех видов (Valid/Dubious).
   - Извлечение уникальных синонимов.
   - Построение таксономического древа предков через API шаблонов.
4. Валидация: сопоставление статусов видов с общим списком родов.
5. Синхронизация: проверка и исправление ярусов на основе возраста (Ma).
6. Сборка БД: создание SQLite базы с тремя связанными таблицами.
